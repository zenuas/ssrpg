## ラグランジュ3

ラグランジュ3は防御がしっかりしていれば全ての敵の攻撃を完全防御できる。
宙域1で楽に突破可能。

### 単艦突破例

```
旗艦: 超重改修艦ハーシェル
主砲: 宙域制圧ミサイル1
チップ: 対火防御x1、対電磁防御x1、自動装填x8、実弾錬成術x25、自動回復遅延x15
```
