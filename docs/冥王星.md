## 冥王星

宇宙空母を旗艦に、強襲揚陸艦を僚艦に、特化SF-AS01を艦載機にできれば比較的楽にクリアできる。
四次元格納庫チップは非常に重要なため取得できれば最優先で開発する。

### 真破 単艦突破例

冥王星は旗艦防御力が1,999,600あれば全ての敵の攻撃を完全防御できる。
宙域1で楽に突破可能。

```
旗艦: 超重改修艦ハーシェル
主砲: 宙域制圧ミサイル1
チップ: 対火防御x1、対電磁防御x1、自動装填x8、実弾錬成術x25、自動回復遅延x15
```
