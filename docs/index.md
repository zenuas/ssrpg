## トップページ

スマホゲームの宇宙戦艦物語RPGの攻略情報メモです。  
主に真破攻略のための情報です。  


なるべく集めやすい武装での攻略例にしていますが、
主要艦種熟練度はカンスト前提、
武装Lvも特に断りがなければカンスト前提です。(旗艦ならLv10,000、僚艦ならなるべく多め)
