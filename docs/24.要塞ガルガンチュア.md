## 要塞ガルガンチュア

統合軍宇宙戦艦や隕石要塞を旗艦に、重病院船・汎用SF-AS21を僚艦に、重輸送艦を艦載機にするとよい。
特にうまみはないステージのため、クリアだけを目指す。

### 一般兵 突破例

重病院船、重輸送艦を配備すればボス以外は大した脅威ではない。
ボスは大きいため、艦載機を突撃させる。

```
旗艦: 青版超級空母
主砲: 無し
副砲: 無し
弾幕: 22mm機銃座
機関: 原子炉H
艦載機: 重輸送艦
チップ: 時空制御x1、四次元格納庫チップx7
僚艦: 宇宙戦闘機X2x1、青版超級空母x3、ミサイル艦x3、重病院船x4、重輸送艦x5
兵種: 功績値獲得x10、弾幕攻撃力x40、対ダメージx145、ボス設計図x97、資金x80、味方対ダメx275
```

### 真破 単艦突破例

要塞ガルガンチュアは旗艦防御力が1,999,600あれば全ての敵の攻撃を完全防御できる。
宙域1で楽に突破可能。

```
旗艦: 超重改修艦ハーシェル
主砲: 宙域制圧ミサイル1
チップ: 対火防御x1、対電磁防御x1、自動装填x8、実弾錬成術x25、自動回復遅延x15
```
