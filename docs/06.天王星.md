## 天王星

兵種の弾幕攻撃力や対ダメージを上げれば攻略できる。
エネルギー不足であれば兵種のバリア回復も上げていく。
ファフニール級重空母などがドロップする宇宙空母の設計図を入手し旗艦としたい。

重力震推進チップは有用だがクリアするのがやっとという事であれば、後で回収するほうがよい。

### 真破 単艦突破例

天王星は旗艦防御力が1,999,600あれば全ての敵の攻撃を完全防御できる。
宙域1で楽に突破可能。

```
旗艦: 超重改修艦ハーシェル
主砲: 宙域制圧ミサイル1
チップ: 対火防御x1、対電磁防御x1、自動装填x8、実弾錬成術x25、自動回復遅延x15
```
