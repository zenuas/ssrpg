## 連星ディオス

重宇宙空母を旗艦に、重病院船・汎用SF-AS21を僚艦に、重輸送艦を艦載機にするとよい。
ボスの覚醒者用SFオーディーンは宙域制圧ミサイル1(基礎攻撃力1500)、オーディーンソード(基礎攻撃力999)と高威力兵器を連発してくる。
防御力を上げていなければ、まず被害は免れない。

ボスは超弩級戦艦メタトロンの設計図をドロップするので開発し旗艦としたいが、開発費が非常に高い。
メタトロンが旗艦として強いのは積載量が大きいからである。
メタトロン開発の際には艦載機に強襲SF-AS400を載せたい。異次元銀河のフォーマルハウトでドロップできる。
アウレリウス恒星系で取得可能な艦載機しかなければメタトロンは不要である。

### 真破 単艦突破例

連星ディオスは旗艦防御力が1,999,600あれば全ての敵の攻撃を完全防御できる。
宙域1で楽に突破可能。

```
旗艦: 超重改修艦ハーシェル
主砲: 宙域制圧ミサイル1
チップ: 対火防御x1、対電磁防御x1、自動装填x8、実弾錬成術x25、自動回復遅延x15
```
