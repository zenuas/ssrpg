## サルガッソー宙域

裏ボスのフレイヤが超重SAフレイヤの設計図をドロップするため優先して取得する。

ボスのフレイは実弾カット90%、Eカット50.9%、回避率60%で実弾が有利となる。
裏ボスのフレイヤは実弾・Eカット共に95%、回避率80%である。

### 真破 単艦突破例

サルガッソー宙域は旗艦防御力が1,999,600あれば全ての敵の攻撃を完全防御できる。
宙域1で楽に突破可能。

```
旗艦: 超重改修艦ハーシェル
主砲: 宙域制圧ミサイル1
チップ: 自動装填x8、実弾錬成術x25、自動回復遅延x17
```
