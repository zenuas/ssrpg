## 惑星ヒュプノス

重宇宙空母を旗艦に、宇宙空母を僚艦に、特化SF-AS01を艦載機にするとよい。
ボスの第六艦隊旗艦アイスサードの大質量氷塊弾(基礎攻撃力2000)はくらうと即撃沈となるため距離をとって相殺したい。

ボスは爆風を当てると[資源](資源.md#爆風資源ドロップ)をドロップするが、これで資源を稼ぐのはあまりに非効率なため推奨しない。
[序盤攻略](序盤攻略.md)にもあるようにアウレリウス恒星系までは僚艦、艦載機でクリアし、
[資源](資源.md)にあるメタルSFで稼ぐ方が何倍も速く資源が貯まる。

### 真破 単艦突破例

惑星ヒュプノスは旗艦防御力が1,999,600あれば雑魚敵の攻撃を完全防御できる。
宙域1で楽に突破可能。

大質量氷塊弾のダメージは `基礎攻撃力2000 * 威力倍率補正1001 - 旗艦防御力1,999,600` となり2400しかくらわない。
防御艦1隻で完全防御できるが、防御艦なしで食らってもバリアで十分カバーできる。

```
旗艦: 超重改修艦ハーシェル
主砲: 宙域制圧ミサイル1
チップ: 対火防御x1、対電磁防御x1、自動装填x8、実弾錬成術x25、自動回復遅延x15
```
