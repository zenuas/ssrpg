## 宇宙闘技場

宇宙闘技場は単艦限定である。
ランクに応じた敵を撃破すればランクが上がる。
負けるとランクが5、または[偽装ランクチップ](チップ.md#偽装ランク)のLv分下がり、資金が `資金 - {資金 / 10}` になる。

### ランク補正

宇宙闘技場に登場する敵は下記表のステータスか機体性能の優秀な方が使用される。(一部例外あり)

|----------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

### 乱入

宇宙闘技場では `max(10, 35 - 超次元ビーコン)分の1` の確率で乱入が発生する。
超次元ビーコンなしでも35分の1、最大10分の1の確率で乱入が発生する。乱入を防止する方法はない。
ランクによって乱入してくる敵が異なる。
登場時のランクによってランク補正を受ける。

| ランク   | 乱入機体               |
|----------|------------------------|
| 99以下   | 資金運搬船             |
| 100～199 | 特化SF極改             |
| 200～299 | 覚醒者用SFオーディーン |
| 300以上  | 強襲SF-AS400GB         |

### ドロップ率

難易度を選択することはできず、敵のLvはランクに応じて変動する。
内部的には難易度 一般兵となるため武装や機関や設計図のドロップ数は1となる。
ただし、[ドロップ率](武装ドロップ.md)は高い。

全てステージボス扱いになるため設計図追加ドロップ判定と資金2倍ドロップが発生する。
複数体同時出現する敵の場合は最後に撃破した敵のみがステージボス扱いになる。
ランク466 黄道騎士アクエリアス(41体出現)からの設計図ドロップ期待値は `41体 * 16.16% + 設計図追加ドロップ` となり7.6256となる。

41体出現する場合のドロップ数期待値は以下の通りとなる。
[ボス設計図Lv](兵種.md#ボス設計図)は97あるものとする。

|------------------------|-----------------:|-----------------------:|
| 主砲・副砲・弾幕・機関 |           6.6256 |                 9.1102 |
| 資金                   |          42.0000 |                42.0000 |
| 救出ポッド             |          41.0000 |                41.0000 |
| 設計図                 |           7.6256 |                 7.6256 |
| 装甲回復ドローン       |           0.8282 |                 3.3128 |

## 強襲SF-AS400GB 攻略例

特に強力な強襲SF-AS400GBの攻略例を紹介する。

### 被ダメージ

強襲SF-AS400GBは強力な武装と高い補正を持つ。
旗艦防御力が1,999,600あっても弾幕しか完全防御できない。

主砲のダメージは `(基礎攻撃力2500 * 威力倍率補正1001 - 旗艦防御力1,999,600 + ランク加算466) * 発射数8 / 装填1秒` で4,026,928ダメージ毎秒となる。
副砲のダメージは `(基礎攻撃力8500 * 威力倍率補正1001 - 旗艦防御力1,999,600 + ランク加算466) * 発射数1 / 装填0.4秒` で16,273,415ダメージ毎秒となる。
合計で20,300,343ダメージ毎秒である。

旗艦回避率を最大の90%とすると2,029,545ダメージ毎秒まで軽減できる。
主砲は散布界が高く全弾命中するわけではないため、おおむね30秒間で5000万～8000万程度の被ダメージとなる。
[敵発射妨害系チップ](チップ.md#敵発射妨害系チップ)を最大までつけると被ダメージを半分程度まで軽減できる。

強襲SF-AS400GBの武装は貫通力がそれほどなく、弾速も遅いため[自動装填](チップ.md#自動装填)と[斬鉄剣](チップ.md#斬鉄剣)と[迎撃弾幕強化](チップ.md#迎撃弾幕強化)を併用すれば
おおむね30秒間でノーダメージ～200万程度の被ダメージまで軽減できる。
接近されると弾破壊できず被ダメージが増えてしまうため、画面端に追い込まれないよう注意したい。

### 光子フィン

真破では重量が重すぎて使えないが、なんでもありならDPSランキング上位に君臨する光子フィンで攻略。
Ver1.0.8で未来視チップが導入されたことで主砲が大幅強化された。
それでも2000万～3000万の装甲は必要になるものの、自機移動なしで秒殺可能。

```
旗艦: 重病院船
主砲: 光子フィン(Lv=10000)

### メガフレアコア

副砲は威力が劣るものが多いものの、比較的DPSランキング上位に位置するメガフレアコアで攻略。
Ver1.0.9で未来視チップが導入されたことで副砲も大幅強化された。
それでも5000万～6000万の装甲は必要になるものの、自機移動なしで秒殺可能。

```
旗艦: 重病院船
副砲: メガフレアコア(Lv=10000)
チップ: 全盛り
