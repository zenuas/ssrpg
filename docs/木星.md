## 木星

敵の火力が上がってくるので僚艦の被害が大きくなってくる。
艦載機はいくらやられても資金や人口の被害はないため、できるだけ艦載機を前衛に出し敵の数を減らしたい。
兵種の弾幕攻撃力、対ダメージを上げて挑戦したい。

### 真破 単艦突破例

木星は旗艦防御力が1,999,600あれば全ての敵の攻撃を完全防御できる。
宙域1で楽に突破可能。

```
旗艦: 超重改修艦ハーシェル
主砲: 宙域制圧ミサイル1
チップ: 対火防御x1、対電磁防御x1、自動装填x8、実弾錬成術x25、自動回復遅延x15
```
