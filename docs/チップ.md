### AUTO性能アップ

照準を自動設定にした場合、一定時間ごとに現在のターゲットに向かって照準をセットする。
AUTO性能アップを設定した場合は `敵スピード * AUTO性能アップLv` 時間ごとに前回の照準位置を記憶し照準を調整している。
ただし、ターゲットとの位置ベクトルを正規化したのち `照準 * 2 - 前回照準位置` としている。
敵を撃破してターゲットが移るなど大きくターゲットが変わったときにずれやすい。

正規化したベクトルの2倍から別の正規化したベクトルを引く、その意味は理解できない。
上記の式の意味が理解できるならつければよい。


### 自動装填

旗艦の装填時間を `装填時間 - 装填時間 * 自動装填Lv / 100` にする。
装填時間の最小値は0.1である。


### 超時空警報、オトリ大作戦、重力震推進、時空制御、宇宙狼煙

超時空警報は1エリア毎の敵出現上限数を増やす。
オトリ大作戦は敵出現間隔を増やす。
宇宙狼煙は敵出現間隔を減らす。
効率に関しては[撃破数稼ぎ](撃破数稼ぎ.md)参照。

時空制御は指定倍速でゲームを進行する。
重力震推進は1秒間に進行する航行距離を伸ばす。
短縮効果に関しては[机の引き出し周回](机の引き出し周回.md)参照。


### メイン弾道計算、サブ弾道計算、Bモード射撃

メイン弾道計算、サブ弾道計算は散布界が `散布界 * (100 - 弾道計算Lv) / 100` になる。
Bモード射撃は散布界が `散布界 + Bモード射撃Lv` になる。

散布界が上がるとランダムに飛ぶ方向が変わるが、このゲームにおいて散布界は射出角度ではない。
弾の進行方向に対するx軸、y軸成分のランダムな加算値(-散布界 ～ +散布界のランダム値を2.8で割ったもの)とされているため、弾の速度が変わる。


### E消費効率化、超E消費効率化

エネルギー消費が減る。
E消費効率化のみが99の場合の最小値は元のE消費の1パーセントを切り上げたものになる。
超E消費効率化を最大まで上げると最小値はおおよそ元のE消費の0.01パーセントを切り上げたものになる。
最低E消費は1残る。

```
E消費2 = E消費1 - {E消費1 / 100 * 超E}
E消費1 = E消費 - {E消費 / 100 * E消費効率化Lv}
超E = {超E消費効率化Lv / 10}

※E消費効率化Lvが99かつ超E消費効率化Lvが1以上の場合のみE消費2の計算が行われる
※超Eの最小値は1となる

式中の{ }は小数点切り捨て
```

### 超キャッチ光線

アイテムの基本速度は0.3である。
超キャッチ光線はアイテムの移動速度を `0.3 + 超キャッチ光線Lv * 0.1` にする。


### 擬似覚醒モード

旗艦の移動スピードを `擬似覚醒モードLv * 0.01` 上昇させる。
チップの説明文に単艦突破時とあるが単艦かどうかにかかわらず移動速度は上昇する。


### 超次元ビーコン

超次元ビーコンに振ると裏ボスが出現するようになる。
以下の条件をすべて満たすと裏ボス出現となる。
超次元ビーコンは2000までしか意味がない。
超次元ビーコンに2000振るか、クリア回数が1000回以上になると50%の確率で裏ボスが出現するようになる。

* 超次元ビーコンが1以上
* 1000分の(クリア回数+超次元ビーコン/2)の抽選
* 固定50%の抽選
* (クリア回数+超次元ビーコン/2)が41以上

### 重撃化系チップ

### 偽装ランク

通常宇宙闘技場で敗北するとランクが5下がるが、偽装ランクを使用すると偽装ランクLv分下げることができる。
偽装ランクを1だけつければ、敗北時のランクは1しか下がらない。


### 戦略的撤退

戦略的撤退Lvの距離まで航行すると自動的に撤退する。
[机の引き出し周回](机の引き出し周回.md)や[撃破数稼ぎ](撃破数稼ぎ.md)の効率アップに用いることができる。


### 自動回復遅延

敵艦の自動回復を自動回復遅延Lv秒遅らせることができる。
自動回復遅延チップでは自動回復を遅らせることはできても止めることはできない。
基本的には終盤ほど重要になるが、[帝国軍移民船団](10.帝国軍移民船団.md)の真破だけは付けすぎるとクリア不可になる事がある。


### 敵発射妨害系チップ

敵の弾装填時間を `装填時間 * (1 + 敵発射妨害系チップLv / 300)` にする。
最大まで振ると敵の装填時間が倍(攻撃が半分)になる。


### サイクロプス

要塞砲の貫通力をあげると説明されているが、貫通力を上げるチップではない。
通常、弾が敵にヒットした際は弾の与ダメージから敵のバリア、装甲分だけ引いて貫通していく。
サイクロプスは要塞砲が敵にヒットした際に与ダメージを `サイクロプス * 0.001` パーセント分切り捨てし回復するチップである。


与ダメージの70%以下の敵にヒットした場合は、貫通力の続く限り射程限界まで貫通する。
```
要塞砲の与ダメージが10,000で、バリア1,000、装甲5,000の敵に貫通ヒットするとする。

サイクロプスがない場合は
1体目の敵にバリアへ1,000ダメージ、装甲に5,000ダメージを与え、残り与ダメージ4,000で貫通する。
2体目の敵にバリアへ1,000ダメージ、装甲に3,000ダメージを与え、弾は消滅する。

サイクロプスが700ある場合は
1体目の敵にバリアへ1,000ダメージ、装甲に5,000ダメージを与え、「残り与ダメージ4,000 + [元与ダメージ10,000 * サイクロプス700 * 0.001]」で与ダメージが10,000まで完全回復し貫通する。
2体目以降も当たるごとに完全回復し貫通し続ける。
```

与ダメージの70%超の敵艦にヒットした場合、トータルで3～4倍程度のダメージを与える。
最終的に与ダメージが100以下になると弾は消滅する。
```
要塞砲の与ダメージが10,000で、バリア1,000、装甲8,000の敵に貫通ヒットするとする。

サイクロプスがない場合は
1体目の敵にバリアへ1,000ダメージ、装甲に8,000ダメージを与え、残り与ダメージ1,000で貫通する。
2体目の敵にバリアへ1,000ダメージを与え、弾は消滅する。

サイクロプスが700ある場合は
1体目の敵にバリアへ1,000ダメージ、装甲に8,000ダメージを与え、「残り与ダメージ1,000 + [元与ダメージ10,000 * サイクロプス700 * 0.001]」で与ダメージが8,000まで回復し貫通する。
2体目の敵にバリアへ1,000ダメージ、装甲に7,000ダメージを与え、「残り与ダメージ0 + [元与ダメージ8,000 * サイクロプス700 * 0.001]」で与ダメージが5,600まで回復し貫通する。
3体目の敵にバリアへ1,000ダメージ、装甲に4,600ダメージを与え、「残り与ダメージ0 + [元与ダメージ5,600 * サイクロプス700 * 0.001]」で与ダメージが3,920まで回復し貫通する。
4体目の敵にバリアへ1,000ダメージ、装甲に2,920ダメージを与え、「残り与ダメージ0 + [元与ダメージ2,920 * サイクロプス700 * 0.001]」で与ダメージが2,044まで回復し貫通する。
5体目の敵にバリアへ1,000ダメージ、装甲に1,044ダメージを与え、「残り与ダメージ0 + [元与ダメージ2,044 * サイクロプス700 * 0.001]」で与ダメージが1,430まで回復し貫通する。
6体目の敵にバリアへ1,000ダメージ、装甲に430ダメージを与え、「残り与ダメージ0 + [元与ダメージ1,430 * サイクロプス700 * 0.001]」で与ダメージが1,001まで回復し貫通する。
7体目の敵にバリアへ1,000ダメージ、装甲に1ダメージを与え、「残り与ダメージ0 + [元与ダメージ1,001 * サイクロプス700 * 0.001]」で与ダメージが700まで回復し貫通する。
8体目の敵にバリアへ700ダメージを与え、「残り与ダメージ0 + [元与ダメージ700 * サイクロプス700 * 0.001]」で与ダメージが490まで回復し貫通する。
9体目の敵にバリアへ490ダメージを与え、「残り与ダメージ0 + [元与ダメージ490 * サイクロプス700 * 0.001]」で与ダメージが343まで回復し貫通する。
10体目の敵にバリアへ343ダメージを与え、「残り与ダメージ0 + [元与ダメージ343 * サイクロプス700 * 0.001]」で与ダメージが240まで回復し貫通する。
11体目の敵にバリアへ240ダメージを与え、「残り与ダメージ0 + [元与ダメージ240 * サイクロプス700 * 0.001]」で与ダメージが168まで回復し貫通する。
12体目の敵にバリアへ168ダメージを与え、「残り与ダメージ0 + [元与ダメージ168 * サイクロプス700 * 0.001]」で与ダメージが117まで回復し貫通する。
13体目の敵にバリアへ117ダメージを与え、「残り与ダメージ0 + [元与ダメージ117 * サイクロプス700 * 0.001]」で与ダメージが81まで回復するが、残り与ダメージが100以下のため消滅する。
```

未強化のベータ要塞砲(基礎威力15)を装備しサイクロプスを付けると、与ダメージが100以下の判定が優先され、通常貫通するはずであっても即消滅する事態となる。


### 斬鉄剣、弾幕迎撃強化

斬鉄剣はソード系武装の、弾幕迎撃強化は対空機銃系武装の弾破壊力をチップLv分上昇する。


### 射撃指揮能力+系チップ

僚艦、艦載機の装填時間を `装填時間 - 装填時間 * 射撃指揮能力+Lv / 1000` にする。
装填時間の最小値は0.1である。
