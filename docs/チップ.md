### チップ入手条件

チップはステージクリア時に難易度に応じた確率で入手できる。
ステージクリアを重ねると入手確率が上がる。

| 条件                 | チップ入手確率                    | 最低入手確率 |
|----------------------|-----------------------------------|-------------:|
| 戦闘訓練場(全難易度) | 101分の(101 + ステージクリア回数) |      100.00% |
| 一般兵、宇宙闘技場   | 101分の(6 + ステージクリア回数)   |        5.94% |
| 熟練兵               | 101分の(16 + ステージクリア回数)  |       15.84% |
| 強化兵               | 101分の(31 + ステージクリア回数)  |       30.69% |
| 親衛隊               | 101分の(76 + ステージクリア回数)  |       75.24% |
| 覚醒者、光化、真破   | 101分の(101 + ステージクリア回数) |      100.00% |

ステージクリア回数は難易度ごとのクリア回数が参照される。
ステージ挑戦時点のクリア回数を参照する。
射爆場、主砲射爆場、試剣場、弾幕射爆場は1000体の標的を撃破時に確定入手できる。


### AUTO性能アップ

照準を自動設定にした場合、一定時間ごとに現在のターゲットに向かって照準をセットする。
AUTO性能アップを設定した場合は `敵スピード * AUTO性能アップLv` 時間ごとに前回の照準位置を記憶し照準を調整している。
ターゲットとの位置ベクトルを正規化したのち `照準 * 2 - 前回照準位置` としている。
敵を撃破してターゲットが移るなど大きくターゲットが変わったときにずれやすい。

正規化したベクトルの2倍から別の正規化したベクトルを引く、その意味は理解できない。
上記の式の意味が理解できるならつければよい。


### 対火電防御

旗艦の火災、電磁障害の回避率を上げる。
対火災力、対電磁力の最大値は99である。
全艦最低でも対火災力、対電磁力が1あるため、対火電防御Lv98まで上げると火災、電磁障害が発生しなくなる。


### 自動装填

旗艦の装填時間を `装填時間 - 装填時間 * 自動装填Lv / 100` にする。
装填時間の最小値は0.1である。


### 超時空警報、オトリ大作戦、重力震推進、時空制御、宇宙狼煙

超時空警報は1エリア毎の敵出現上限数を超時空警報Lv分増やす。
オトリ大作戦は敵出現間隔を `オトリ大作戦 / 10` 増やす。
宇宙狼煙は敵出現間隔を `1 - 宇宙狼煙 / 20` 倍に減らす。
効率に関しては[撃破数稼ぎ](撃破数稼ぎ.md)参照。

時空制御は指定倍速でゲームを進行する。
重力震推進は1秒間に進行する航行距離を `1 + 重力震推進Lv / 50` にする。
短縮効果に関しては[机の引き出し周回](机の引き出し周回.md)参照。


### 圧縮弾倉、超圧縮弾倉

旗艦の弾数を `圧縮弾倉Lv + 超圧縮弾倉Lv` パーセント増やす。
小数点以下は切り上げされる。
弾数の上限は9999発。


### 四次元格納庫

艦載機数が上昇する。
艦載機が搭載不可な艦には効果がない。


### 短距離跳躍

短距離跳躍Lv分開始ステージをスキップする。
ステージ数が7未満のエリアと太陽では無効となる。


### メイン弾道計算、サブ弾道計算、Bモード射撃

メイン弾道計算、サブ弾道計算は散布界が `散布界 * (100 - 弾道計算Lv) / 100` になる。
Bモード射撃は散布界が `散布界 + Bモード射撃Lv` になる。

散布界が上がるとランダムに飛ぶ方向が変わるが、このゲームにおいて散布界は射出角度ではない。
弾の進行方向に対するx軸、y軸成分のランダムな加算値(-散布界 ～ +散布界のランダム値を2.8で割ったもの)とされているため、弾の速度が変わる。


### E消費効率化、超E消費効率化

エネルギー消費が減る。
E消費効率化のみが99の場合の最小値は元のE消費の1パーセントを切り上げたものになる。
超E消費効率化を最大まで上げると最小値はおおよそ元のE消費の0.01パーセントを切り上げたものになる。
最低E消費は1残る。

```
E消費2 = E消費1 - {E消費1 / 100 * 超E}
E消費1 = E消費 - {E消費 / 100 * E消費効率化Lv}
超E = {超E消費効率化Lv / 10}

※E消費効率化Lvが99かつ超E消費効率化Lvが1以上の場合のみE消費2の計算が行われる
※超Eの最小値は1となる

式中の{ }は小数点切り捨て
```


### 貫通強化系チップ

貫通力がチップLv分増加する。
装備画面では主砲実弾貫通強化、副砲実弾貫通強化しか表示されないが正しく反映されている。


### 超キャッチ光線

アイテムの基本速度は0.9である。
超キャッチ光線はアイテムの移動速度を `0.9 + 超キャッチ光線Lv * 0.3` にする。


### 慣性制御ソフトApple3

旗艦、僚艦、艦載機の旋回速度が `慣性制御ソフトApple3Lv * 0.1` 上がる。


### 擬似覚醒モード

旗艦の移動スピードを `擬似覚醒モードLv * 0.03` 上昇させる。
チップの説明文に単艦突破時とあるが単艦かどうかにかかわらず移動速度は上昇する。


### 実弾錬成術

10秒毎に実弾錬成術Lvパーセントの弾数を回復する。
小数点以下は切り捨てされる。
最低でも1は回復する。


### 超次元ビーコン

超次元ビーコンに振ると裏ボスが出現するようになる。
以下の条件をすべて満たすと裏ボス出現となる。
超次元ビーコンは2000までしか意味がない。
超次元ビーコンに2000振るか、クリア回数が1000回以上になると50%の確率で裏ボスが出現するようになる。
超次元ビーコンがメタルSFなどの出現率を上げるというのはデマである。

* 超次元ビーコンが1以上
* 1000分の(クリア回数 + 超次元ビーコン / 2)の抽選
* 固定50%の抽選
* (クリア回数 + 超次元ビーコン / 2)が41以上

また、[宇宙闘技場](12.宇宙闘技場.md)では `(35 - 超次元ビーコン)分の1` の確率で[乱入](12.宇宙闘技場.md#乱入)が発生する。
超次元ビーコンなしでも35分の1、最大でも10分の1の確率で乱入が発生する。

### 偽装ランク

通常宇宙闘技場で敗北するとランクが5下がるが、偽装ランクを使用すると偽装ランクLv分下げることができる。
偽装ランクを1だけつければ、敗北時のランクは1しか下がらない。


### 戦略的撤退

戦略的撤退Lvの距離まで航行すると自動的に撤退する。
[机の引き出し周回](机の引き出し周回.md)や[撃破数稼ぎ](撃破数稼ぎ.md)の効率アップに用いることができる。


### 自動回復遅延

敵艦の自動回復を自動回復遅延Lv秒遅らせることができる。
自動回復遅延チップでは自動回復を遅らせることはできても止めることはできない。
基本的には終盤ほど重要になるが、[帝国軍移民船団](10.帝国軍移民船団.md)の真破だけは付けすぎるとクリア不可になる事がある。


### 敵発射妨害系チップ

敵の弾装填時間を `装填時間 * (1 + 敵発射妨害系チップLv / 300)` にする。
最大まで振ると敵の装填時間が倍(攻撃が半分)になる。


### サイクロプス

要塞砲の貫通力をあげると説明されているが、貫通力を上げるチップではない。
通常、弾が敵にヒットした際は弾の与ダメージから敵のバリア、装甲分だけ引いて貫通していく。
サイクロプスは要塞砲が敵にヒットした際に与ダメージを `サイクロプス * 0.1` パーセント分切り捨てし回復し、最終的に与ダメージが100以下になると弾は消滅する。
実際のダメージに関しては[貫通ダメージについて](ダメージ計算.md#貫通ダメージについて)参照。

### 射撃指揮能力+系チップ

僚艦、艦載機の装填時間を `装填時間 - 装填時間 * 射撃指揮能力+Lv / 1000` にする。
装填時間の最小値は0.1である。


### 軽量化系チップ

武装、機関の重量が `重量 * 軽量化Lv / 1000` 減らす。
小数点以下は切り上げされる。


### 実弾近接化系チップ

実弾の残存時間を `実弾近接化Lv * 0.05` 減らす。
残存時間の最小値は0.1である。

ゲーム内の[射程表記](その他.md#射程と全艦射程)は `弾速 * 残存時間` であるため減らすメリットは少ない。
爆発するタイプの実弾武装において爆風を手前で発生させることができる。
爆発しないタイプの武装にはメリットはない。


### 艦隊運動の名人

旗艦、僚艦、艦載機の速度が `艦隊運動の名人Lv * 0.03` に固定される。
[擬似覚醒モード](#擬似覚醒モード)の上昇分は無視される。
旗艦に限れば艦隊運動の名人を付けずに、擬似覚醒モードだけ付ける方が早い。

### 未来視系チップ

敵の回避率を `回避率 - [未来視Lv * 0.1]` にする。
回避率を減少させるのは実弾やエネルギー兵器のみで爆風回避率には適用されない。
