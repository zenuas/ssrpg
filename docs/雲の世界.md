### クリア数について

総合クリア数の上限は `太陽以外の48エリア * 難易度7段階 + 太陽光化 + 射爆場 + 主砲射爆場 + 試剣場 + 弾幕射爆場 + 宇宙闘技場` の計342となる。
[太陽一般兵出撃バグ](バグ.md#太陽一般兵出撃バグ)を利用した場合、総合クリア数上限は343となる。

単艦クリア数の上限は `単艦出撃可能な47エリア * 難易度7段階 + 太陽光化 + 射爆場 + 主砲射爆場 + 試剣場 + 弾幕射爆場 + 宇宙闘技場` の計335となる。
[ネオ・コロニー](43.ネオ・コロニー.md)、[ブルクムーン回廊](51.ブルクムーン回廊.md)、[泡立つ宇宙](52.泡立つ宇宙.md)、[スターチャイルド](53.スターチャイルド.md)の真破はその難易度から単艦クリアはできないと考えられ、単艦クリア数の上限は331となる。
[太陽一般兵出撃バグ](バグ.md#太陽一般兵出撃バグ)を利用した場合、単艦クリア数上限は332となる。
[小惑星ベヒモス](11.小惑星ベヒモス.md)は仕様上単艦クリアができない。

ダミークリア数の上限は `太陽以外の41エリア * 難易度7段階 + 太陽光化` の計288となる。
[太陽一般兵出撃バグ](バグ.md#太陽一般兵出撃バグ)と[スコーピオ再出撃バグ](バグ.md#スコーピオ再出撃バグ)を利用した場合、ダミークリア数上限は293となる。
[小惑星ベヒモス](11.小惑星ベヒモス.md)、[ディアスポラ回廊](32.ディアスポラ回廊.md)、[ベルセルク](39.ベルセルク.md)、[惑星サーズ](40.惑星サーズ.md)、[オルフェウス](42.オルフェウス.md)、[ネオ・コロニー](43.ネオ・コロニー.md)、[スコーピオ](44.スコーピオ.md)、宇宙遊技場関連は仕様上ダミークリアができない。

射爆場、主砲射爆場、試剣場、弾幕射爆場は初回の的2体を倒した時点でクリアとして扱われる。
[宇宙闘技場](12.宇宙闘技場.md)はランク1の敵を倒した時点でクリアとして扱われる。
